Funo para Benchmark - R
trials <- function(action, iterations) {
tempos <- numeric()
for (i in 1:iterations) {
tempo1 <- Sys.time()
action()
tempo2 <- Sys.time()
tempos <- c(tempos,as.numeric(tempo2-tempo1))
}
print(mean(tempos))
print(sd(tempos))
}

function trials(action::Function, iterations::Int)
tempos = Float64[]

tempo1 = time()

tempo2 = time()
push!(tempos, tempo2 - tempo1)
end

println(std(tempos))
end

Arquivo no formato Parquet
Os arquivos Parquet so um formato de armazenamento de dados colunar que
oferece alta eficincia para trabalhar com grandes volumes de informaes, j
que apenas as colunas relevantes para uma determinada anlise precisam ser
lidas do disco, em vez de todo o conjunto de dados.
Diferente dos formatos tradicionais como CSV que armazenam dados linha por
linha, o Parquet organiza as informaes por colunas, o que significa que todos
os valores de uma mesma coluna so armazenados juntos.

Arquivo no formato Json
JSON um formato de arquivo leve e totalmente independente de
linguagem para armazenar e transmitir dados. Apesar do nome fazer
referncia ao JavaScript, ele universal e suportado pela maioria das
linguagens de programao modernas.
A estrutura do JSON baseada em dois conceitos fundamentais: Pares
de Chave-Valor e Listas Ordenadas.

com grandes volumes de dados.
